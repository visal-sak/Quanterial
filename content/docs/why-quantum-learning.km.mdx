វេទិកាជាច្រើនគ្រាន់តែផ្ដល់វីដេអូឱ្យអ្នក ហើយសង្ឃឹមថានឹងបានលទ្ធផលល្អ។ Quantum Learning ត្រូវបានបង្កើតឡើង
ជុំវិញគំនិតមួយចំនួនដែលមានភស្តុតាងគាំទ្រ អំពីរបៀបដែលមនុស្សពិតជារៀន — ហើយអ្វីៗគ្រប់យ៉ាង
នៅក្នុងផលិតផលនេះ ដើរតាមគំនិតទាំងនោះ។

## បញ្ហាដែលយើងកំណត់ដើម្បីដោះស្រាយ

- **ការរៀនតាមបែបអកម្មមិនចាប់ជាប់ទេ។** ការមើល និងការអានឡើងវិញ មានអារម្មណ៍ថាមានផលិតភាព ប៉ុន្តែ
  រសាត់បាត់យ៉ាងលឿន។
- **វិធីមួយសម្រាប់គ្រប់គ្នាធ្វើឱ្យខ្ជះខ្ជាយពេលវេលា។** អ្នកអង្គុយស្ដាប់រឿងដែលអ្នកចេះរួច ឬ
  វង្វេងក្នុងរឿងដែលអ្នកមិនចេះ។
- **ការញាត់រៀនឈ្នះការរៀនមានចន្លោះ — រហូតដល់ថ្ងៃប្រឡង។** បន្ទាប់មកវាបាត់អស់។
- **គ្មានវដ្តមតិត្រឡប់ទេ។** បើគ្មានការដឹង *ហេតុអ្វី* អ្នកខុស កំហុសនឹងកើតឡើងម្ដងទៀត។

## គោលការណ៍ដែលយើងផ្អែកលើ

### ១. ការរំលឹកសកម្ម (Active recall)

ការទាញយកព័ត៌មានចេញមកវិញ ពង្រឹងការចងចាំច្រើនជាងការពិនិត្យវាឡើងវិញច្រើន។ ដូច្នេះរាល់មេរៀន
ធ្វើឱ្យអ្នក *បង្កើត* ចម្លើយ មិនមែនគ្រាន់តែស្គាល់ពួកវាទេ។ សូមមើល
[ការអនុវត្ត និងមតិត្រឡប់](/km/docs/practice)។

### ២. ការធ្វើម្ដងទៀតមានចន្លោះ (Spaced repetition)

ការពិនិត្យឡើងវិញនៅចន្លោះពេលដែលកើនឡើងបន្តិចម្ដងៗ — ពិតជាមុនពេលដែលអ្នកនឹងភ្លេច — មានប្រសិទ្ធភាព
ខ្លាំងជាងការអនុវត្តជាកម្រៃច្រើន។ គំរូ [ការស្ទាត់ជំនាញ](/km/docs/mastery) របស់យើងកំណត់កាលវិភាគនេះឱ្យ
អ្នកដោយស្វ័យប្រវត្តិ។

### ៣. ភាពសម្របតាម (Adaptivity)

ពេលវេលាគឺជាធនធានកម្រនិងមានតម្លៃបំផុតរបស់អ្នក។ [ផ្លូវសិក្សា](/km/docs/learning-paths) រំលងអ្វីដែលអ្នក
ចេះរួច ហើយផ្ដោតលើអ្វីដែលអ្នកមិនចេះ។

### ៤. មតិត្រឡប់ភ្លាមៗ ដែលមានការពន្យល់

មតិត្រឡប់ដែលមកដល់ភ្លាមៗ និងពន្យល់ពី *ហេតុផល* បម្លែងរាល់កំហុសទៅជា
មេរៀន។

## របៀបដែលគោលការណ៍ទាំងនេះបង្ហាញនៅក្នុងផលិតផល

| គោលការណ៍          | កន្លែងដែលអ្នកឃើញវា                                  |
| ------------------ | ------------------------------------------------- |
| ការរំលឹកសកម្ម      | ការអនុវត្តបន្ទាប់ពីរាល់គោលគំនិត                       |
| ការធ្វើម្ដងទៀតមានចន្លោះ  | ជួរការរំលឹកប្រចាំថ្ងៃ                             |
| ភាពសម្របតាម         | កម្រងសំណួរដាក់កម្រិត + ការកាត់តម្រឹមផ្លូវសិក្សា                     |
| មតិត្រឡប់ដែលមានការពន្យល់ | ការបង្ហាញ "ហេតុអ្វី" បន្ទាប់ពីរាល់ចម្លើយ                |

## វាសម្រាប់នរណា

Quantum Learning ដំណើរការសម្រាប់អ្នកចាប់ផ្ដើមរៀនដោយខ្លួនឯង សិស្សនិស្សិត អ្នកប្ដូរអាជីព និង
អ្នកជំនាញដែលចង់ស្រាយឡើងវិញ។ ប្រសិនបើអ្នកចង់ *ចងចាំពិតប្រាកដ* នូវអ្វីដែលអ្នកសិក្សា — មិនមែនគ្រាន់តែ
មើលវារំលងទៅ — នោះអ្នកមកដល់កន្លែងត្រឹមត្រូវហើយ។
