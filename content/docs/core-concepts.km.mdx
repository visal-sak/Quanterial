ការយល់ដឹងអំពីគោលគំនិតស្នូលមួយចំនួន ធ្វើឲ្យវេទិកាទាំងមូលមានតម្លាភាព។ ទំព័រនេះគឺជា
គំរូគំនិត (mental model) នៅពីក្រោយអ្វីៗគ្រប់យ៉ាងផ្សេងទៀតនៅក្នុងឯកសារ — អានវាម្តង
រួចផ្នែកដែលនៅសល់នឹងមានលក្ខណៈច្បាស់លាស់។

## រង្វិលជុំនៃការសិក្សា

រាល់ផ្នែកនីមួយៗនៃ Quantum Learning ត្រូវបានស្ថាបនាឡើងជុំវិញរង្វិលជុំបួនជំហានដូចគ្នា។
វាធ្វើឡើងវិញនៅគ្រប់មាត្រដ្ឋាន ចាប់ពីសំណួរអនុវត្តតែមួយ រហូតដល់ផ្លូវសិក្សាពេញលេញ។

1. **រៀន** — ស្រូបយកគំនិតផ្តោតតែមួយ។
2. **អនុវត្ត** — យកវាទៅប្រើភ្លាមៗ។
3. **មតិត្រឡប់** — ដឹងភ្លាមៗនូវអ្វីដែលអ្នកធ្វើបានត្រូវ និងខុស។
4. **រំលឹក** — ត្រឡប់មកវិញនៅពេលក្រោយ ដើម្បីឲ្យវាជាប់។

<Callout title="ហេតុអ្វីរង្វិលជុំនេះមានសារៈសំខាន់">
  ការស្រាវជ្រាវអំពីការរំលឹកមានចន្លោះពេល (spaced repetition) និងការនឹកឡើងវិញយ៉ាងសកម្ម
  បង្ហាញថា ការ*ទាញយក*ចំណេះដឹងវិញ ល្អជាងការ*អានឡើងវិញ*។ រង្វិលជុំនេះត្រូវបានរចនាឡើង
  ដើម្បីឲ្យអ្នកទាញយកឆាប់ និងញឹកញាប់។
</Callout>

## មេរៀន

**មេរៀន** គឺជាឯកតាតូចបំផុតនៃការសិក្សា៖ គំនិតផ្តោតតែមួយ ដែលពន្យល់ដោយសង្ខេប
ហើយពង្រឹងដោយការអនុវត្ត។ មេរៀនត្រូវបានរចនាឡើងឲ្យអាចបញ្ចប់បានក្នុងការអង្គុយម្តង
ប្រហែល ៥–១០ នាទី។

មេរៀនធម្មតាមួយមានៈ

- ការពន្យល់ខ្លីៗ (អត្ថបទ ដ្យាក្រាម ឬវីដេអូ)។
- ឧទាហរណ៍ដែលបានធ្វើរួចមួយ ឬច្រើន។
- សំណួរអនុវត្តមួយចំនួន។
- ការសង្ខេបភ្លាមៗនូវអ្វីដែលអ្នកទើបតែបានរៀន។

```ts
// The shape of a lesson in the content model
interface Lesson {
  id: string;
  title: string;
  estimatedMinutes: number;
  concepts: string[]; // topics this lesson strengthens
  practice: Question[];
}
```

## វគ្គសិក្សា

**វគ្គសិក្សា** គឺជាការប្រមូលផ្តុំមេរៀនតាមលំដាប់ជុំវិញប្រធានបទតែមួយ។ ការបញ្ចប់
វគ្គសិក្សាមួយ មានន័យថាអ្នកបានធ្វើការឆ្លងកាត់មេរៀនរាល់មេរៀន ហើយបានឆ្លងកាត់
ការត្រួតពិនិត្យអនុវត្តរបស់វា។

| លក្ខណៈវគ្គសិក្សា | អត្ថន័យ                                          |
| --------------- | ------------------------------------------------ |
| `title`         | អ្វីដែលវគ្គសិក្សាបង្រៀន                          |
| `lessons`       | បញ្ជីមេរៀនតាមលំដាប់                             |
| `level`         | `beginner`, `intermediate`, ឬ `advanced`        |
| `outcomes`      | អ្វីដែលអ្នកនឹងអាចធ្វើបាននៅពេលអ្នកបញ្ចប់        |

វគ្គសិក្សាមានលក្ខណៈពេញលេញដោយខ្លួនឯង។ អ្នកអាចរៀនវាតែឯង ឬឲ្យ
[ផ្លូវសិក្សា](/km/docs/learning-paths) រៀបលំដាប់ជាច្រើនរួមគ្នាឲ្យអ្នក។

## ផ្លូវសិក្សា

**ផ្លូវសិក្សា** ភ្ជាប់វគ្គសិក្សាច្រើនចូលគ្នាទៅជាដំណើរឆ្ពោះទៅរកគោលដៅធំជាង —
ឧទាហរណ៍ "ពីអ្នកចាប់ផ្តើមថ្មីរហូតដល់ត្រៀមខ្លួនធ្វើការ។" ផ្លូវសិក្សាសម្របតាមអ្វី
ដែលអ្នកបានដឹងស្រាប់ ដូច្នេះអ្នកមិនចាំបាច់អង្គុយឆ្លងកាត់សម្ភារៈដែលអ្នកបានស្ទាត់ជំនាញ
រួចហើយឡើយ។

## ការស្ទាត់ជំនាញ

រាល់ការព្យាយាមអនុវត្តនីមួយៗ ធ្វើបច្ចុប្បន្នភាពពិន្ទុ **ការស្ទាត់ជំនាញ** (mastery)
របស់អ្នកសម្រាប់ប្រធានបទមួយ — ជាលេខរវាង ០ និង ១ ដែលប៉ាន់ស្មានថាអ្នកស្គាល់វាបានល្អ
កម្រិតណា។ ការស្ទាត់ជំនាញថយចុះបន្តិចម្តងៗតាមពេលវេលា ដូច្នេះការរំលឹកម្តងម្កាល
រក្សាចំណេះដឹងឲ្យស្រស់។ ការស្ទាត់ជំនាញត្រូវបានពន្យល់លម្អិតនៅលើទំព័ររបស់វាផ្ទាល់៖
[ការស្ទាត់ជំនាញ និងវឌ្ឍនភាព](/km/docs/mastery)។

```ts
// Fetch a learner's mastery for a topic
const mastery = await quantum.mastery.get({ topic: 'algebra-basics' });
console.log(`Mastery: ${Math.round(mastery.score * 100)}%`);
```

## របៀបដែលគោលគំនិតទាំងនេះភ្ជាប់គ្នា

នេះជាឋានានុក្រមចាប់ពីតូចបំផុតទៅធំបំផុតៈ

| គោលគំនិត    | វិសាលភាព             | តាមដាន                     |
| ---------- | ----------------- | -------------------------- |
| សំណួរ   | ការត្រួតពិនិត្យមួយ         | ត្រូវ / ខុស        |
| មេរៀន     | គំនិតមួយ          | ការបញ្ចប់                 |
| វគ្គសិក្សា   | ប្រធានបទមួយ         | មេរៀនដែលបានបញ្ចប់          |
| ផ្លូវសិក្សា       | គោលដៅមួយ          | វគ្គសិក្សាដែលបានបញ្ចប់            |
| ការស្ទាត់ជំនាញ    | ជំនាញមួយតាមពេលវេលា | កម្លាំងនៃចំណេះដឹងរបស់អ្នក |

<Callout title="គួរដឹង" type="info">
  ការស្ទាត់ជំនាញគឺតាមប្រធានបទ មិនមែនតាមវគ្គសិក្សាឡើយ។ ការរំលឹកមេរៀនមួយ អាចបង្កើន
  ការស្ទាត់ជំនាញដែលវគ្គសិក្សាជាច្រើនចែករំលែក — ដែលនោះជាមូលហេតុដែលវេទិកាពេលខ្លះ
  ណែនាំមេរៀនមួយពីវគ្គសិក្សាដែលអ្នកមិនទាន់បានចាប់ផ្តើមជាផ្លូវការ។
</Callout>
