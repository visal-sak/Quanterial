Advanced paths move faster and go deeper. They assume you're comfortable with the
[core concepts](/docs/core-concepts) and want to build real, portfolio-ready skills.

## Who advanced paths are for

- Learners who've finished a beginner path or already know the basics.
- People preparing for a specific goal: a job, an interview, or a project.
- Anyone who learns best by building real things.

## Popular advanced paths

### Applied Data Analysis

Go from spreadsheets to real analysis: statistics, modeling, and working with messy,
real-world datasets.

- **Length:** ~5 courses
- **Time:** 12–15 hours total
- **Capstone:** analyze a real dataset and present your findings.

### Ship a Full Project

Plan, build, and deploy something end to end. You'll practice the whole lifecycle,
not just isolated skills.

- **Length:** ~6 courses
- **Time:** 15–20 hours total
- **Capstone:** a deployed project you can share.

### Interview Ready

Targeted practice for technical interviews — problem solving, communication, and
timed challenges.

- **Length:** ~4 courses
- **Time:** 10–12 hours total
- **Capstone:** a full mock interview with feedback.

## What to expect

- Project-based lessons with fewer hand-holds.
- Optional stretch challenges for extra depth.
- Mastery checks that unlock the next stage.
- Realistic, open-ended problems instead of toy examples.

## Staying challenged

Advanced paths use your [mastery](/docs/mastery) scores aggressively. If you breeze
through the early checks, the path fast-forwards you to harder material so you're
never bored.

```ts
// Advanced paths raise the difficulty as mastery climbs
function nextDifficulty(mastery: number) {
  if (mastery > 0.9) return 'stretch';
  if (mastery > 0.7) return 'standard';
  return 'foundational';
}
```

<Callout title="Skip what you know">
  The placement quiz can fast-forward you past material you've already mastered so
  advanced paths stay challenging. Not sure you're ready? Try a
  [Beginner Path](/docs/learning-paths/beginner) first.
</Callout>
