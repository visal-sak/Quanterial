ផ្លូវសិក្សាកម្រិតខ្ពស់ដំណើរការលឿនជាង ហើយចុះជ្រៅជាង។ វាសន្មតថាអ្នកមានភាពស្រួលជាមួយ
[គំនិតគ្រឹះ](/km/docs/core-concepts) ហើយចង់ស្ថាបនាជំនាញពិតប្រាកដ ដែលអាចដាក់ក្នុងផតហ្វូលីយោ (portfolio) បាន។

## ផ្លូវសិក្សាកម្រិតខ្ពស់សម្រាប់អ្នកណា

- អ្នករៀនដែលបានបញ្ចប់ផ្លូវសិក្សាកម្រិតដំបូង ឬចេះមូលដ្ឋានគ្រឹះរួចហើយ។
- មនុស្សដែលកំពុងរៀបចំសម្រាប់គោលដៅជាក់លាក់៖ ការងារ ការសម្ភាសន៍ ឬគម្រោង។
- អ្នកណាដែលរៀនបានល្អបំផុតតាមរយៈការស្ថាបនាអ្វីៗពិតប្រាកដ។

## ផ្លូវសិក្សាកម្រិតខ្ពស់ដ៏ពេញនិយម

### ការវិភាគទិន្នន័យអនុវត្ត

ចេញពីសៀវភៅបញ្ជី (spreadsheets) ទៅរកការវិភាគពិតប្រាកដ៖ ស្ថិតិ ការធ្វើគំរូ និងការធ្វើការជាមួយ
សំណុំទិន្នន័យពិភពលោកពិតដ៏រញ៉េរញ៉ៃ។

- **ប្រវែង៖** ~5 វគ្គសិក្សា
- **ពេលវេលា៖** 12–15 ម៉ោងសរុប
- **គម្រោងបញ្ចប់ (Capstone)៖** វិភាគសំណុំទិន្នន័យពិត និងបង្ហាញលទ្ធផលរបស់អ្នក។

### ដឹកជញ្ជូនគម្រោងពេញលេញ

រៀបចំផែនការ ស្ថាបនា និងដាក់ដំណើរការ (deploy) អ្វីមួយពីដើមដល់ចប់។ អ្នកនឹងអនុវត្តវដ្តជីវិតទាំងមូល
មិនមែនត្រឹមតែជំនាញដាច់ដោយឡែកនោះទេ។

- **ប្រវែង៖** ~6 វគ្គសិក្សា
- **ពេលវេលា៖** 15–20 ម៉ោងសរុប
- **គម្រោងបញ្ចប់ (Capstone)៖** គម្រោងដែលបានដាក់ដំណើរការ ដែលអ្នកអាចចែករំលែកបាន។

### ត្រៀមខ្លួនសម្រាប់ការសម្ភាសន៍

ការអនុវត្តដែលមានគោលដៅសម្រាប់ការសម្ភាសន៍បច្ចេកទេស — ការដោះស្រាយបញ្ហា ការទំនាក់ទំនង និង
ការប្រកួតដែលកំណត់ពេលវេលា។

- **ប្រវែង៖** ~4 វគ្គសិក្សា
- **ពេលវេលា៖** 10–12 ម៉ោងសរុប
- **គម្រោងបញ្ចប់ (Capstone)៖** ការសម្ភាសន៍សាកល្បងពេញលេញ ជាមួយមតិកែលម្អ។

## អ្វីដែលត្រូវរំពឹងទុក

- មេរៀនផ្អែកលើគម្រោង ជាមួយការជួយផ្គូនតិចជាង។
- បញ្ហាប្រកួតបន្ថែម (stretch challenges) ជាជម្រើស សម្រាប់ជម្រៅបន្ថែម។
- ការត្រួតពិនិត្យការស្ទាត់ជំនាញ ដែលដោះសោដំណាក់កាលបន្ទាប់។
- បញ្ហាជាក់ស្ដែង បើកចំហ ជំនួសឱ្យឧទាហរណ៍ក្មេងលេង។

## រក្សាភាពប្រកួតប្រជែង

ផ្លូវសិក្សាកម្រិតខ្ពស់ប្រើពិន្ទុ[ការស្ទាត់ជំនាញ](/km/docs/mastery)របស់អ្នកយ៉ាងខ្លាំង។ បើអ្នកឆ្លងកាត់
ការត្រួតពិនិត្យដំបូងៗយ៉ាងស្រួល ផ្លូវសិក្សានឹងបញ្ជូនអ្នកទៅមុខ ទៅកាន់មេរៀនពិបាកជាង ដូច្នេះអ្នក
មិនដែលធុញទ្រាន់ឡើយ។

```ts
// Advanced paths raise the difficulty as mastery climbs
function nextDifficulty(mastery: number) {
  if (mastery > 0.9) return 'stretch';
  if (mastery > 0.7) return 'standard';
  return 'foundational';
}
```
