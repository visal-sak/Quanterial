**ផ្លូវសិក្សា** គឺជាលំដាប់នៃវគ្គសិក្សាដែលបានរៀបចំយ៉ាងម៉ត់ចត់ ដែលនាំអ្នកចេញពីកន្លែងដែលអ្នក
កំពុងស្ថិតនៅ ទៅកាន់កន្លែងដែលអ្នកចង់ទៅ។ ជំនួសឱ្យការស្មានថាត្រូវរៀនអ្វីបន្ទាប់ ចូរដើរតាមផ្លូវមួយ
ហើយទុកឱ្យវេទិកាគ្រប់គ្រងលំដាប់ ល្បឿន និងការត្រួតពិនិត្យឡើងវិញជំនួសអ្នក។

## ហេតុអ្វីត្រូវប្រើផ្លូវសិក្សា ជំនួសឱ្យការជ្រើសរើសវគ្គសិក្សាដោយខ្លួនឯង?

- **គ្មានការនឿយហត់ក្នុងការសម្រេចចិត្ត** — អ្នកតែងតែដឹងអ្វីតែមួយដែលត្រូវធ្វើបន្ទាប់។
- **លំដាប់ត្រឹមត្រូវ** — មុខវិជ្ជាមុនតម្រូវ (prerequisites) មកមុនអ្វីៗដែលត្រូវការវា។
- **ល្បឿនសម្របតាមស្ថានភាព** — ប្រធានបទដែលអ្នកចេះរួចហើយត្រូវបានកាត់ចេញដោយស្វ័យប្រវត្តិ។
- **បន្ទាត់បញ្ចប់** — គោលដៅច្បាស់លាស់ និងវិញ្ញាបនបត្រនៅពេលអ្នកបញ្ចប់វា។

## ផ្លូវសិក្សាដំណើរការយ៉ាងដូចម្ដេច

1. **ជ្រើសរើសគោលដៅ** — ឧ. "មូលដ្ឋានគ្រឹះនៃការវិភាគទិន្នន័យ។"
2. **ទទួលបានផែនការ** — យើងរៀបចំវគ្គសិក្សាត្រឹមត្រូវសម្រាប់គោលដៅនោះ។
3. **រៀនតាមការសម្រប** — ប្រធានបទដែលអ្នកចេះរួចហើយត្រូវបានរំលង ឬកាត់ឱ្យខ្លី។
4. **តាមដានវឌ្ឍនភាព** — ផ្ទាំងគ្រប់គ្រងរបស់អ្នកបង្ហាញមេរៀនបន្ទាប់ដែលបានផ្ដល់អនុសាសន៍។
5. **បញ្ចប់ដោយទំនុកចិត្ត** — ទទួលបានវិញ្ញាបនបត្រនៅពេលផ្លូវសិក្សាបានបញ្ចប់។

## រចនាសម្ព័ន្ធនៃផ្លូវសិក្សា

ផ្លូវសិក្សាគ្រាន់តែជាបញ្ជីវគ្គសិក្សាដែលបានរៀបលំដាប់ បូកនឹងសេចក្ដីថ្លែងអំពីគោលដៅ៖

```ts
const path = {
  id: 'data-analysis-fundamentals',
  goal: 'Analyze real datasets with confidence',
  level: 'beginner',
  courses: [
    'spreadsheets-basics',
    'intro-to-statistics',
    'data-visualization',
    'telling-stories-with-data',
  ],
};
```

## ការសម្របដំណើរការយ៉ាងដូចម្ដេច

មុនពេលអ្នកចាប់ផ្ដើម **កម្រងសំណួរដាក់កម្រិត (placement quiz)** ជាជម្រើសមួយ វាស់ស្ទង់
[ការស្ទាត់ជំនាញ](/km/docs/mastery) ដែលអ្នកមានរួចហើយលើប្រធានបទក្នុងផ្លូវសិក្សានេះ។ អ្វីៗដែលអ្នកបានស្ទាត់ជំនាញរួច
ត្រូវបានកាត់ចេញ ដូច្នេះផ្លូវសិក្សានៅតែផ្ដោតលើអ្វីដែលថ្មីសម្រាប់អ្នក។

| ការស្ទាត់ជំនាញរបស់អ្នកលើប្រធានបទ | អ្វីដែលផ្លូវសិក្សាធ្វើ           |
| ----------------------- | ---------------------------- |
| ខ្ពស់ (≥ 80%)            | រំលងប្រធានបទទាំងស្រុង      |
| មធ្យម (40–80%)         | បង្ហាញតែការរំឮកខ្លីៗ  |
| ទាប (< 40%)             | រួមបញ្ចូលវគ្គសិក្សាពេញលេញ      |

## ការតាមដានផ្លូវសិក្សា

ផ្ទាំងគ្រប់គ្រងរបស់អ្នកបង្ហាញវឌ្ឍនភាពសរុបនៃផ្លូវសិក្សា បូកនឹងមេរៀនបន្ទាប់ដែលបានផ្ដល់អនុសាសន៍ ដូច្នេះអ្នក
តែងតែដឹងអ្វីតែមួយដ៏ល្អបំផុតដែលត្រូវធ្វើបន្ទាប់។

```ts
const progress = await quantum.paths.progress('data-analysis-fundamentals');
console.log(`${progress.completedCourses}/${progress.totalCourses} courses done`);
console.log('Next up:', progress.nextLesson.title);
```

<Callout title="សម្របតាមការរចនា">
  ធ្វើកម្រងសំណួរដាក់កម្រិតមុនពេលចាប់ផ្ដើមផ្លូវសិក្សា — ការស្ទាត់ជំនាញដែលអ្នកមានរួចហើយ អនុញ្ញាតឱ្យយើង
  កាត់ផ្លូវសិក្សា ដើម្បីឱ្យអ្នកចំណាយពេលតែលើអ្វីដែលថ្មីប៉ុណ្ណោះ។
</Callout>

## វិញ្ញាបនបត្រ

បញ្ចប់រាល់វគ្គសិក្សាទាំងអស់ក្នុងផ្លូវសិក្សា ហើយអ្នកនឹងទទួលបាន **វិញ្ញាបនបត្រ** ដែលអ្នកអាចចែករំលែកបាន។ វិញ្ញាបនបត្រនីមួយៗ
រៀបរាប់ពីគោលដៅ វគ្គសិក្សាដែលបានបញ្ចប់ និងកាលបរិច្ឆេទ — ជាមធ្យោបាយសាមញ្ញមួយក្នុងការ
បង្ហាញនូវអ្វីដែលអ្នកបានរៀន។
