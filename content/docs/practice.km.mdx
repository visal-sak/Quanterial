ការអានមិនមែនជាការសិក្សាឡើយ — ការ**ទាញយកឡើងវិញ**ទេទើបជាការសិក្សា។ Quantum Learning
ត្រូវបានស្ថាបនាឡើងដើម្បីឲ្យអ្នកអនុវត្តឥតឈប់ឈរ ហើយទទួលបានមតិត្រឡប់នៅពេលដែលអ្នកត្រូវការ។

## ប្រភេទនៃការអនុវត្ត

ជំនាញផ្សេងៗគ្នាត្រូវការការអនុវត្តប្រភេទផ្សេងគ្នា ដូច្នេះមេរៀនលាយបញ្ចូលទម្រង់ជាច្រើនៈ

| ទម្រង់            | ល្អបំផុតសម្រាប់                          |
| ----------------- | --------------------------------- |
| ជម្រើសច្រើន   | ការស្គាល់គោលគំនិតយ៉ាងលឿន       |
| បំពេញចន្លោះ | ការនឹកឡើងវិញនូវពាក្យ ឬវាក្យសម្ព័ន្ធពិតប្រាកដ    |
| ចម្លើយខ្លី      | ការពន្យល់គំនិតតាមពាក្យរបស់អ្នកផ្ទាល់ |
| អន្តរកម្ម/កូដ  | ការយកជំនាញទៅអនុវត្តដោយផ្ទាល់ដៃ            |
| អូស និងទម្លាក់     | ការរៀបលំដាប់ជំហាន ឬការផ្គូផ្គងគូ   |

## មតិត្រឡប់ភ្លាមៗ

បន្ទាប់ពីរាល់ការព្យាយាមនីមួយៗ អ្នកទទួលបានមតិត្រឡប់ភ្លាមៗ។ មតិត្រឡប់ល្អធ្វើរឿងបីយ៉ាងៈ

1. **ប្រាប់អ្នកថាតើអ្នកត្រូវឬអត់។**
2. **ពន្យល់ពីមូលហេតុ** — ការវែកញែក មិនមែនត្រឹមតែចម្លើយឡើយ។
3. **ចង្អុលទៅមុខ** — អ្វីដែលត្រូវរំលឹក ប្រសិនបើអ្នកឆ្លើយខុស។

```ts
interface Feedback {
  correct: boolean;
  explanation: string;   // why this is the answer
  reviewTopics?: string[]; // what to revisit if you missed it
}
```

<Callout title="ចម្លើយខុសមានប្រយោជន៍">
  ការឆ្លើយសំណួរខុស គឺជាសញ្ញាសិក្សាដ៏ខ្លាំងបំផុតមួយដែលមាន។ វេទិកាផ្តល់ទម្ងន់ខ្លាំង
  ដល់កំហុសដែលបាន*កែតម្រូវ* នៅពេលធ្វើបច្ចុប្បន្នភាព
  [ការស្ទាត់ជំនាញ](/km/docs/mastery) របស់អ្នក — ដូច្នេះកុំខ្លាចក្នុងការស្មាន និងរៀន។

## ការណែនាំ មិនមែនចម្លើយ

ជាប់គាំង? សុំ **ការណែនាំ** (hint) ជំនួសឲ្យការបង្ហាញចម្លើយ។ ការណែនាំរុញអ្នកឆ្ពោះទៅ
ជំហានបន្ទាប់ ខណៈពេលរក្សាការងារ "ទាញយកឡើងវិញ" ដ៏មានតម្លៃ ឲ្យនៅជាការងាររបស់អ្នកធ្វើ។

## កម្រិតលំបាកសម្របតាមអ្នក

កម្រិតលំបាកនៃការអនុវត្ត តាមដានការស្ទាត់ជំនាញរបស់អ្នក។ នៅពេលអ្នករីកចម្រើន សំណួរកាន់តែ
ពិបាក បើអ្នកមានការលំបាក វេទិកានឹងបន្ធូរបន្ថយ ហើយពង្រឹងគ្រឹះមូលដ្ឋានជាមុនសិន។

```ts
function pickDifficulty(mastery: number) {
  if (mastery > 0.85) return 'hard';
  if (mastery > 0.5) return 'medium';
  return 'easy';
}
```

## ការរំលឹកមានចន្លោះពេល

សំណួរដែលអ្នកបានឆ្លើយខុស — ឬមិនបានឃើញអស់រយៈពេលមួយ — លេចឡើងម្តងទៀតនៅក្នុង **ជួរ
ការរំលឹកប្រចាំថ្ងៃ** របស់អ្នក នៅគ្រាដែលអ្នកទំនងជាកំពុងភ្លេចវាបំផុត។ ការចំណាយ
ប៉ុន្មាននាទីក្នុងមួយថ្ងៃ រក្សាអ្វីៗគ្រប់យ៉ាងឲ្យស្រស់។
