## គំនិត

**ការជាប់ទាក់ទងគ្នា (entanglement)** ភ្ជាប់គុយប៊ីតពីរ ដើម្បីឲ្យលទ្ធផលនៃការវាស់របស់ពួកវាមានទំនាក់ទំនងគ្នា —
មិនថាពួកវានៅឆ្ងាយពីគ្នាប៉ុនណានោះទេ។ ការវាស់មួយ ប្រាប់អ្នកភ្លាមៗនូវអ្វីមួយអំពី
មួយទៀត។

សភាពជាប់ទាក់ទងគ្នាដ៏ល្បីមួយគឺ **Bell state**៖

```
|Φ⁺⟩ = 0.707 |00⟩ + 0.707 |11⟩
```

នេះមានន័យថា៖ គុយប៊ីតទាំងពីរ **តែងតែដូចគ្នា**។ បើគុយប៊ីតទីមួយវាស់បាន `0`,
គុយប៊ីតទីពីរគឺ `0`; បើទីមួយជា `1`, ទីពីរជា `1` — ប៉ុន្តែថាតើអ្នកនឹងបានមួយណានោះ គឺ
ចៃដន្យ។

## អ្វីដែលការជាប់ទាក់ទងគ្នា *មិនមែន*

<Callout title="គ្មានការផ្ញើសារលឿនជាងពន្លឺ" type="warn">
  ការជាប់ទាក់ទងគ្នា **មិន** អនុញ្ញាតឲ្យអ្នកផ្ញើព័ត៌មានលឿនជាងពន្លឺទេ។ ភាគីនីមួយៗឃើញ
  លទ្ធផលចៃដន្យ; ទំនាក់ទំនងលេចឡើងតែនៅពេលដែលអ្នកប្រៀបធៀបកំណត់ត្រាតាមរយៈ
  ឆានែលធម្មតា (ដែលយឺតជាងពន្លឺ)។
</Callout>

## ឧទាហរណ៍ដែលបានដោះស្រាយ

```ts
// Measuring the first qubit of a Bell state determines the second
function measureBell(firstResult: 0 | 1) {
  const secondResult = firstResult; // always matches in |Φ⁺⟩
  return { firstResult, secondResult };
}

console.log(measureBell(1)); // { firstResult: 1, secondResult: 1 }
```

| គុយប៊ីតទីមួយវាស់បាន    | គុយប៊ីតទីពីរក្លាយជា    |
| -------------------- | -------------------- |
| 0                    | 0                    |
| 1                    | 1                    |

## ការអនុវត្ត

**សំណួរ៖** គុយប៊ីតពីរស្ថិតនៅក្នុង Bell state `|Φ⁺⟩`។ អ្នកវាស់ទីមួយ ហើយបាន
`0`។ តើគុយប៊ីតទីពីរនឹងជាអ្វី?

<details>
  <summary>បង្ហាញចម្លើយ</summary>

  **0។** នៅក្នុង `|Φ⁺⟩` គុយប៊ីតមានទំនាក់ទំនងគ្នាយ៉ាងល្អឥតខ្ចោះ ដូច្នេះទីពីរដូចនឹង
  ទីមួយ។
</details>
