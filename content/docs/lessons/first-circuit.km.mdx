## គំនិត

ដល់ពេលបញ្ចូលអ្វីៗគ្រប់យ៉ាងគ្នា។ **សៀគ្វី quantum** គឺជាលំដាប់នៃ
[ច្រក](/km/docs/lessons/gates) ដែលដាក់ទៅលើគុយប៊ីតតាមពេលវេលា — អានពីឆ្វេងទៅស្តាំ ដូចជា
បទភ្លេង។ នៅក្នុងមេរៀននេះ យើងនឹងសង់សៀគ្វីដែលបង្កើត
[ជាប់ទាក់ទងគ្នា (entanglement)](/km/docs/lessons/entanglement) **Bell state**។

## រូបមន្ត

គ្រាន់តែច្រកពីរនៅលើគុយប៊ីតពីរ (ទាំងពីរចាប់ផ្តើមនៅ `|0⟩`)៖

1. ដាក់ច្រក **H** ទៅគុយប៊ីត 0 → វាចូលទៅក្នុងសភាពត្រួតគ្នា។
2. ដាក់ **CNOT** ជាមួយគុយប៊ីត 0 ជាច្រកបញ្ជា និងគុយប៊ីត 1 ជាគោលដៅ → ទាំងពីរក្លាយជា
   ជាប់ទាក់ទងគ្នា។

លទ្ធផលគឺ Bell state `|Φ⁺⟩ = 0.707|00⟩ + 0.707|11⟩`។

## ជំហានម្តងមួយៗ

| ជំហាន | សកម្មភាព        | សភាពនៃប្រព័ន្ធ                    |
| ---- | ------------- | -------------------------------- |
| 0    | ចាប់ផ្តើម       | `|00⟩`                           |
| 1    | H នៅលើគុយប៊ីត 0 | `0.707|00⟩ + 0.707|10⟩`          |
| 2    | CNOT (0 → 1)  | `0.707|00⟩ + 0.707|11⟩`          |

## ក្នុងកូដ

```ts
// Pseudocode for a Bell-state circuit
const circuit = createCircuit({ qubits: 2 });

circuit.h(0);        // superposition on qubit 0
circuit.cnot(0, 1);  // entangle qubit 1 with qubit 0

const result = circuit.measureAll();
// result is "00" or "11", each ~50% of the time — never "01" or "10"
console.log(result);
```

<Callout title="អ្នកទើបតែសង់ការជាប់ទាក់ទងគ្នា">
  ដំណើរការសៀគ្វីនេះច្រើនដង នោះអ្នកនឹងឃើញតែ `00` ឬ `11` ប៉ុណ្ណោះ។ ទំនាក់ទំនង
  ដ៏ល្អឥតខ្ចោះនោះ គឺជាការជាប់ទាក់ទងគ្នាកំពុងដំណើរការ — គោលបំណងទាំងមូលនៃមេរៀនទី ៣ ដែលឥឡូវនេះ
  ត្រូវបានបង្កើតឡើងតាំងពីដំបូង។
</Callout>

## ការអនុវត្ត

**សំណួរ៖** បន្ទាប់ពីដំណើរការសៀគ្វី Bell state, តើលទ្ធផលនៃការវាស់មួយណាដែល
**មិនអាចទៅរួច**?

<details>
  <summary>បង្ហាញចម្លើយ</summary>

  **`01` និង `10`។** Bell state `|Φ⁺⟩` មានតែ `|00⟩` និង `|11⟩` ប៉ុណ្ណោះ ដូច្នេះគុយប៊ីតទាំង
  ពីរតែងតែដូចគ្នា។
</details>
