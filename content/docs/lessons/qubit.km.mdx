## бһӮбҹҶбһ“бһ·бһҸ

**бһ”бҹҠбһёбһҸ** бһ”бһ»бһҡбһ¶бһҺ бһӮбһәбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғ `0` бһ¬ `1`бҹ” **бһӮбһ»бһҷбһ”бҹҠбһёбһҸ (qubit)** (бһ”бҹҠбһёбһҸquantum) бһўбһ¶бһ…бһҮбһ¶ `0`, `1`,
бһ¬ вҖ” бһўбһҡбһӮбһ»бһҺбһҠбһӣбҹӢ [бһҹбһ—бһ¶бһ–бһҸбҹ’бһҡбһҪбһҸбһӮбҹ’бһ“бһ¶ (superposition)](/km/docs/lessons/superposition) вҖ” бһҖбһ¶бһҡбһӣбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бһ“бҹғбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡ
бһ“бҹ…бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ бһҡбһ бһјбһҸбһҠбһӣбҹӢбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ¶бһҹбҹӢбҹ”

бһҷбһҫбһ„бһҹбһҡбһҹбҹҒбһҡбһҹбһ—бһ¶бһ–бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҹбһүбҹ’бһүбһ¶бһҺ "ket"бҹ–

- `|0вҹ©` вҖ” бһӮбһ»бһҷбһ”бҹҠбһёбһҸбһҮбһ¶ 0 бһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢ
- `|1вҹ©` вҖ” бһӮбһ»бһҷбһ”бҹҠбһёбһҸбһҮбһ¶ 1 бһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢ

## Bloch sphere

бһңбһ¶бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһҡбһҳбҹғбһҳбһҫбһӣбһӮбһ»бһҷбһ”бҹҠбһёбһҸбһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһҳбһҪбһҷбһ“бҹ…бһӣбһҫбһ•бҹ’бһ‘бҹғбһ“бҹғбһҹбҹ’бһңбҹҠбҹӮбһҡ вҖ” **Bloch
sphere** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбҹ” бһ”бҹүбһјбһӣбһҒбһ¶бһ„бһҮбһҫбһ„бһӮбһә `|0вҹ©`, бһ”бҹүбһјбһӣбһҒбһ¶бһ„бһҸбҹ’бһ”бһјбһ„бһӮбһә `|1вҹ©`, бһ бһҫбһҷ
бһҡбһ¶бһӣбҹӢбһ…бҹҶбһҺбһ»бһ…бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһӮбһәбһҮбһ¶бһҹбһ—бһ¶бһ–бһӮбһ»бһҷбһ”бҹҠбһёбһҸбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

| бһӮбҹҶбһ“бһ·бһҸ           | бһ”бҹҠбһёбһҸбһ”бһ»бһҡбһ¶бһҺ      | бһӮбһ»бһҷбһ”бҹҠбһёбһҸ                   |
| -------------- | ------------- | ------------------------ |
| бһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһўбһ¶бһ…бһҳбһ¶бһ“   | 0 бһ¬ 1         | бһ…бҹҶбһҺбһ»бһ…бһҺбһ¶бһҳбһҪбһҷбһ“бҹ…бһӣбһҫбһҹбҹ’бһңбҹҠбҹӮбһҡ       |
| бһҒбһҺбҹҲбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһўбһ¶бһ“    | бһҗбҹҒбһҡ           | бһўбһ¶бһ…бһҮбһ¶бһҖбһ¶бһҡбһӣбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбһ“бҹғбһҹбһ—бһ¶бһ–     |
| бһ–бҹҒбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ¶бһҹбҹӢ   | 0 бһ¬ 1        | бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһ»бҹҮбһ‘бҹ… 0 бһ¬ 1         |

## бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ

бһҖбһ¶бһҡбһңбһ¶бһҹбҹӢбһӮбһ»бһҷбһ”бҹҠбһёбһҸбһҠбҹӮбһӣбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҸбҹ’бһҡбһ„бҹӢбһўбҹҒбһҖбҹ’бһңбһ¶бһ‘бҹҗбһҡ бһ•бҹ’бһҸбһӣбҹӢбһұбһҖбһ¶бһҹ 50/50 бһ“бҹғ `0` бһ¬ `1`бҹ–

```ts
// A qubit on the equator: equal chance of 0 or 1
const qubit = { alpha: 0.707, beta: 0.707 }; // |ПҲвҹ© = 0.707|0вҹ© + 0.707|1вҹ©
const pZero = qubit.alpha ** 2; // 0.5
const pOne = qubit.beta ** 2;   // 0.5
console.log(pZero, pOne); // 0.5 0.5
```

## бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ

**бһҹбҹҶбһҺбһҪбһҡбҹ–** бһӮбһ»бһҷбһ”бҹҠбһёбһҸбһҳбһҪбһҷбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбһ—бһ¶бһ– `|0вҹ©`бҹ” бһҸбһҫбһ”бҹ’бһҡбһјбһ”бһ¶бһ”бҹҠбһёбһӣбһёбһҸбҹҒбһ“бҹғбһҖбһ¶бһҡбһңбһ¶бһҹбҹӢбһ”бһ¶бһ“ `1` бһӮбһәбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“?

<details>
  <summary>бһ”бһ„бҹ’бһ бһ¶бһүбһ…бһҳбҹ’бһӣбһҫбһҷ</summary>

  **0%бҹ”** бһҹбһ—бһ¶бһ– `|0вҹ©` бһӮбҹ’бһҳбһ¶бһ“бһҹбһҳбһ¶бһҹбһ—бһ¶бһӮ `|1вҹ©` бһ‘бҹҒ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҖбһ¶бһҡбһңбһ¶бһҹбҹӢбһҸбҹӮбһ„бһҸбҹӮбһ•бҹ’бһҸбһӣбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣ `0`бҹ”
</details>
