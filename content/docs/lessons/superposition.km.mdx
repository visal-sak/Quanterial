## គំនិត

**សភាពត្រួតគ្នា (superposition)** គឺជាសមត្ថភាពរបស់គុយប៊ីតក្នុងការស្ថិតនៅក្នុងការរួមបញ្ចូលនៃ `|0⟩` និង `|1⟩` នៅ
ពេលតែមួយ។ យើងពិពណ៌នាវាដោយប្រើលេខពីរដែលហៅថា **អំព្លីទុត (amplitude)**៖

```
|ψ⟩ = α|0⟩ + β|1⟩
```

- `α` (alpha) គឺជាអំព្លីទុតសម្រាប់ `0`
- `β` (beta) គឺជាអំព្លីទុតសម្រាប់ `1`

## ច្បាប់នៃល្បែង

នៅពេលអ្នកវាស់ គុយប៊ីតធ្លាក់ចុះទៅ `0` ឬ `1`។ ប្រូបាប៊ីលីតេកើតចេញពី
ការេនៃអំព្លីទុត ហើយពួកវាត្រូវតែបូកបញ្ចូលគ្នាស្មើនឹង 1៖

```
P(0) = |α|²      P(1) = |β|²      |α|² + |β|² = 1
```

| សភាព                       | P(0) | P(1) |
| ------------------------- | ---- | ---- |
| `|0⟩`                     | 100% | 0%   |
| `|1⟩`                     | 0%   | 100% |
| `0.707|0⟩ + 0.707|1⟩`     | 50%  | 50%  |
| `0.6|0⟩ + 0.8|1⟩`         | 36%  | 64%  |

## ឧទាហរណ៍ដែលបានដោះស្រាយ

```ts
function probabilities(alpha: number, beta: number) {
  return { pZero: alpha ** 2, pOne: beta ** 2 };
}

console.log(probabilities(0.6, 0.8)); // { pZero: 0.36, pOne: 0.64 }
```

<Callout title="ហេតុអ្វីបានជាវាសំខាន់">
  សភាពត្រួតគ្នាអនុញ្ញាតឲ្យកុំព្យូទ័រ quantum រុករកលទ្ធភាពជាច្រើនស្របគ្នា។
  សិល្បៈនៃក្បួនដោះស្រាយ quantum គឺការរៀបចំរបស់ផ្សេងៗ ដើម្បីឲ្យចម្លើយ *ត្រឹមត្រូវ* ជាចម្លើយដែលមាន
  លទ្ធភាពខ្ពស់បំផុត នៅពេលដែលអ្នកវាស់ចុងក្រោយ។
</Callout>

## ការអនុវត្ត

**សំណួរ៖** គុយប៊ីតមួយគឺ `0.6|0⟩ + 0.8|1⟩`។ តើប្រូបាប៊ីលីតេនៃការវាស់បាន `0` គឺប៉ុន្មាន?

<details>
  <summary>បង្ហាញចម្លើយ</summary>

  **36%** — ព្រោះថា `P(0) = |0.6|² = 0.36`។
</details>
