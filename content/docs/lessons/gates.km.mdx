## គំនិត

បើគុយប៊ីតគឺជាទិន្នន័យ **ច្រកquantum (quantum gate)** គឺជាប្រតិបត្តិការ។ ច្រកមួយយកគុយប៊ីត
(ឬច្រើន) ហើយបំប្លែងសភាពរបស់វា។ ការភ្ជាប់ច្រកចូលគ្នាបង្កើតបានជា
[សៀគ្វី quantum](/km/docs/lessons/first-circuit)។

## ច្រកដែលអ្នកនឹងជួបជាមុនគេ

| ច្រក  | ឈ្មោះ    | អ្វីដែលវាធ្វើ                                        |
| ----- | -------- | --------------------------------------------------- |
| **X** | NOT      | ត្រឡប់ `|0⟩ ↔ |1⟩`                                    |
| **H** | Hadamard | ដាក់គុយប៊ីតចូលទៅក្នុង [សភាពត្រួតគ្នា (superposition)](/km/docs/lessons/superposition) ស្មើគ្នា |
| **Z** | Phase    | ត្រឡប់សញ្ញានៃផ្នែក `|1⟩`                     |
| **CNOT** | Controlled-NOT | ត្រឡប់គុយប៊ីតគោលដៅ *បើ* ច្រកបញ្ជាគឺ `|1⟩` |

## ច្រក Hadamard

ច្រក **H** គឺជាកម្លាំងចលករនៃ quantum computing។ ដាក់វាទៅ `|0⟩` នោះអ្នកទទួលបាន
គុយប៊ីតដែលជា 50/50៖

```
H|0⟩ = 0.707|0⟩ + 0.707|1⟩
```

```ts
// Applying H to |0⟩ yields an equal superposition
function hadamardOnZero() {
  return { alpha: 0.707, beta: 0.707 }; // 50% / 50%
}
console.log(hadamardOnZero());
```

<Callout title="ច្រកអាចត្រឡប់ក្រោយវិញបាន">
  មិនដូចប្រតិបត្តិការបុរាណជាច្រើនទេ ច្រកquantum អាចត្រូវបានលុបចេញវិញជានិច្ច។ ដាក់ច្រកតែមួយ
  ដូចជា **X** ឬ **H** ពីរដង នោះអ្នកនឹងត្រឡប់ទៅកន្លែងដែលអ្នកបានចាប់ផ្តើមវិញ។
</Callout>

## ការអនុវត្ត

**សំណួរ៖** អ្នកដាក់ច្រក **X** ទៅគុយប៊ីតក្នុងសភាព `|0⟩`។ តើឥឡូវនេះវាស្ថិតនៅក្នុងសភាព
អ្វី?

<details>
  <summary>បង្ហាញចម្លើយ</summary>

  **`|1⟩`។** ច្រក X (NOT) ត្រឡប់ `|0⟩` ទៅ `|1⟩`។
</details>
