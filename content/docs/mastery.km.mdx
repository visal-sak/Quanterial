**ការស្ទាត់ជំនាញ** (mastery) គឺជាលេខតែមួយដែលជំរុញឥរិយាបថឆ្លាតវៃភាគច្រើនរបស់វេទិកា៖
អ្វីដែលត្រូវណែនាំបន្ទាប់ អ្វីដែលត្រូវរំលង និងពេលណាដែលត្រូវរំលឹកអ្នកឲ្យរំលឹកមេរៀន។
ទំព័រនេះពន្យល់ពីរបៀបដែលវាដំណើរការ។

## ការស្ទាត់ជំនាញជាអ្វី

ការស្ទាត់ជំនាញគឺជាពិន្ទុរវាង **០ និង ១** សម្រាប់ **ប្រធានបទ** នីមួយៗ — មិនមែនមេរៀន
ឬវគ្គសិក្សានីមួយៗឡើយ។ វាប៉ាន់ស្មានប្រូបាប៊ីលីតេដែលអ្នកនឹងឆ្លើយសំណួរថ្មីមួយលើ
ប្រធានបទនោះបានត្រូវនៅពេលនេះ។

| ពិន្ទុ       | ការបកស្រាយ                    |
| ----------- | --------------------------------- |
| 0.0 – 0.3   | ថ្មី ឬមិនទាន់រឹងមាំ — ត្រូវការការបង្រៀនពេញលេញ |
| 0.3 – 0.7   | កំពុងរៀន — ត្រូវការការអនុវត្ត          |
| 0.7 – 0.9   | រឹងមាំ — ការរំលឹកម្តងម្កាល           |
| 0.9 – 1.0   | ស្ទាត់ជំនាញ — គ្រាន់តែកែតម្រូវស្រាលៗ     |

## របៀបដែលការស្ទាត់ជំនាញកើនឡើង

រាល់ការព្យាយាមអនុវត្តនីមួយៗ ធ្វើបច្ចុប្បន្នភាពពិន្ទុ៖

- **ចម្លើយត្រូវ** បង្កើនវា — កាន់តែច្រើនសម្រាប់សំណួរពិបាកជាង។
- **ចម្លើយត្រូវ ដែលលឿន និងមានទំនុកចិត្ត** បង្កើនវាបន្តិចទៀត។
- **ការឆ្លើយបានត្រូវ បន្ទាប់ពីធ្លាប់ខកខានពីមុន** ត្រូវបានផ្តល់ទម្ងន់ខ្លាំង
  ព្រោះនោះជាការសិក្សាពិតប្រាកដ។

```ts
// Simplified mastery update
function updateMastery(current: number, correct: boolean, difficulty: number) {
  const target = correct ? 1 : 0;
  const rate = 0.15 * difficulty; // harder questions move the needle more
  return current + (target - current) * rate;
}
```

## របៀបដែលការស្ទាត់ជំនាញថយចុះ

ចំណេះដឹងនឹងរសាត់បាត់ បើអ្នកមិនដែលត្រឡប់មករកវាវិញ ដូច្នេះការស្ទាត់ជំនាញ
**ថយចុះបន្តិចម្តងៗតាមពេលវេលា**។ ការថយចុះមានលក្ខណៈយឺតសម្រាប់ប្រធានបទដែលបានស្ទាត់ជំនាញ
យ៉ាងល្អ ហើយលឿនជាងសម្រាប់ប្រធានបទដែលអ្នកទើបតែបានរៀនថ្មីៗ។

  ការថយចុះនេះជាអ្វីដែលដំណើរការការរំលឹកមានចន្លោះពេល (spaced repetition)។ នៅពេលប្រធានបទ
  មួយធ្លាក់ក្រោមកម្រិតកំណត់ វេទិកានឹងលើកឡើងនូវការរំលឹករហ័សមួយ មុនពេលដែលអ្នកនឹងភ្លេច —
  គ្រាដែលមានប្រសិទ្ធភាពបំផុតដើម្បីពង្រឹងវា។

## វឌ្ឍនភាព ធៀបនឹង ការស្ទាត់ជំនាញ

ពួកវាមានទំនាក់ទំនងគ្នា ប៉ុន្តែខុសគ្នាៈ

- **វឌ្ឍនភាព** = ចំនួនប៉ុន្មាននៃខ្លឹមសារដែលអ្នកបាន*បញ្ចប់* (មេរៀន វគ្គសិក្សា ផ្លូវសិក្សា)។
- **ការស្ទាត់ជំនាញ** = អ្នកពិតជា*ស្គាល់*សម្ភារៈបានល្អកម្រិតណានៅពេលនេះ។

អ្នកអាចមានវឌ្ឍនភាព ១០០% លើវគ្គសិក្សាមួយ ហើយនៅតែមានការស្ទាត់ជំនាញថយចុះនៅ
ប៉ុន្មានសប្តាហ៍ក្រោយ។ ផ្ទាំងគ្រប់គ្រងបង្ហាញទាំងពីរ ដូច្នេះអ្នកទទួលបានរូបភាពពេញលេញ។

## ការរំលឹកយ៉ាងមានប្រសិទ្ធភាព

វេទិកាបង្កើត **ជួរការរំលឹកប្រចាំថ្ងៃ** ពីប្រធានបទដែលការស្ទាត់ជំនាញរបស់វាបានធ្លាក់ចុះ។
ការសម្អាតវាចំណាយពេលត្រឹមតែប៉ុន្មាននាទី ហើយជារឿងដែលមានឥទ្ធិពលខ្ពស់បំផុតដែលអ្នកអាចធ្វើបាន។
