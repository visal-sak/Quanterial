សូមស្វាគមន៍មកកាន់ **Quantum Learning** ដែលជាវេទិកាសិក្សាទំនើប បង្កើតឡើងដើម្បីជួយអ្នក
ឱ្យផ្លាស់ប្ដូរពីភាពចង់ដឹងទៅជាទំនុកចិត្ត។ ឯកសារទាំងនេះនឹងណែនាំអ្នកពីអ្វីៗគ្រប់យ៉ាងដែលអ្នកត្រូវការ ដើម្បីចាប់ផ្ដើម
រៀន តាមដានវឌ្ឍនភាពរបស់អ្នក និងទាញយកអត្ថប្រយោជន៍ច្រើនបំផុតពីវេទិកានេះ។

## តើ Quantum Learning ជាអ្វី?

Quantum Learning រៀបចំចំណេះដឹងជា **មេរៀន** តូចៗ ដែលផ្ដោតច្បាស់លាស់ ដោយដាក់ជាក្រុមក្នុង
**វគ្គសិក្សា (course)** និងរៀបលំដាប់វគ្គសិក្សាទាំងនោះជា **ផ្លូវសិក្សា (learning path)** ដែលមានការណែនាំ។ រាល់មេរៀន
ផ្គូផ្គងការពន្យល់ខ្លីៗជាមួយនឹងការអនុវត្តជាក់ស្ដែង ដូច្នេះអ្នករៀនតាមរយៈការធ្វើ ជាជាងគ្រាន់តែ
អាន។

បន្ទាប់មក វេទិកានេះនឹងតាមដាន **ការស្ទាត់ជំនាញ (mastery)** របស់អ្នកលើប្រធានបទនីមួយៗ ហើយប្រើវាដើម្បីណែនាំ
រឿងតែមួយដ៏ល្អបំផុតដែលអ្នកគួរសិក្សាបន្ទាប់ — ព្រមទាំងរំលឹកអ្នកឱ្យធ្វើការរំលឹកមុននឹងអ្នក
ភ្លេច។

## ហេតុអ្វីអ្នករៀនជ្រើសរើសយើង

| មុខងារ            | អត្ថន័យសម្រាប់អ្នក                                    |
| ------------------ | -------------------------------------------------------- |
| **ខ្នាតតូច**     | មេរៀនចំណាយពេល ៥–១០ នាទី ដូច្នេះវឌ្ឍនភាពសមនឹងកាលវិភាគគ្រប់ពេល |
| **អនុវត្តជាក់ស្ដែង**       | ការអនុវត្តបន្ទាប់ពីរាល់គោលគំនិត ដើម្បីឱ្យវាចាប់ជាប់            |
| **សម្របតាម**       | ផ្លូវសិក្សាកែសម្រួលទៅតាមអ្វីដែលអ្នកចេះរួចហើយ                     |
| **ការរំលឹកមានចន្លោះ**  | យើងលើកយកប្រធានបទឡើងវិញ ពិតជាមុនពេលដែលអ្នកនឹងភ្លេច        |
| **តាមដានវឌ្ឍនភាព** | មើលឃើញ streak ភាពបញ្ចប់ និងការស្ទាត់ជំនាញបានយ៉ាងឆាប់រហ័ស          |

## តើវាដំណើរការជាមួយគ្នាយ៉ាងណា

បំណែកតូចៗ រួមផ្សំគ្នាបង្កើតជាលទ្ធផលធំ៖

**សំណួរ → មេរៀន → វគ្គសិក្សា → ផ្លូវសិក្សា → គោលដៅរបស់អ្នក**

សំណួរបង្កើតមេរៀន មេរៀនបង្កើតវគ្គសិក្សា ហើយវគ្គសិក្សាបង្កើតផ្លូវឆ្ពោះទៅកាន់
គោលដៅរបស់អ្នក — ស្រទាប់នីមួយៗត្រូវបានរៀបលំដាប់ និងកំណត់ល្បឿនសម្រាប់អ្នក។
